# Lidar-Based SLAM & HD Mapping 

## Project Overview
This project focuses on building an **Autonomous Navigation Stack** using **ROS 2 Humble**. The primary goal is to solve **Localization Drift** in unknown environments by fusing Lidar and IMU data.

## Key Features (In Development)
* **Intelligent Navigation:** Implementing SLAM for real-time map generation.
* **Sensor Fusion:** Utilizing an **Extended Kalman Filter (EKF)** to bridge Lidar and IMU inputs.
* **Spatial Logic:** Integrating **Geomatics** principles for high-definition (HD) mapping.

## Technical Stack
* **Language:** C++ (C++17/20)
* **Middleware:** ROS 2 (Humble)
* **Simulation:** Gazebo, RViz
